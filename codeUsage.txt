
1. Open the project folder
   Open Command Prompt or PowerShell and go to the project folder:
   cd d:\MapMyCivic -Web

2. Make sure the required tools are installed
   Install Node.js and npm from https://nodejs.org/
   Install Python 3.10+ from https://www.python.org/

   Check the versions:
   node -v
   npm -v
   python --version

3. Install frontend packages
   Open a terminal and run:
   cd d:\MapMyCivic -Web\frontend
   npm install

4. Install backend packages (FastAPI)
   Open another terminal and run:
   cd d:\MapMyCivic -Web\backend

   Create and activate a virtual environment:
   python -m venv .venv
   .venv\Scripts\activate

   Install Python dependencies:
   pip install -r requirements.txt

   If requirements.txt is not available, install the common packages manually:
   pip install fastapi uvicorn

5. Start the backend (FastAPI)
   In the backend terminal, run:
    uvicorn app.main --host 0.0.0.0 --port 8000 --reload

   If your app is inside a folder, try:
    uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

   If that does not work, check the file name and use the correct entry point.

   The backend should usually run at:
   http://127.0.0.1:8000

   FastAPI documentation will be available at:
   http://127.0.0.1:8000/docs

6. Start the frontend
   In the frontend terminal, run:
   npm run dev

   If that does not work, try:
   npm start

   The frontend usually opens at:
   http://localhost:3000
   or
   http://localhost:5173

7. Stop the apps
   Press Ctrl+C in both terminals to stop the frontend and backend.

8. If you get errors
   - If npm install fails, delete node_modules and package-lock.json, then run npm install again.
   - If Python packages fail to install, make sure the virtual environment is activated.
   - If the backend does not start, check whether the entry file is main.py or app/main.py.
   - If the frontend cannot connect to the backend, make sure the backend is running first and the API URL is correct.

9. Important note
   This project uses FastAPI for the backend, so you must run the backend with Python and Uvicorn instead of using Next.js or other Node.js server commands.
